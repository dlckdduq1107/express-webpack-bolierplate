# Webpack_BoilerPlate

express,babel,source-map,Webpack을 이용한 개발환경 세팅

## npm 설치

    npm install

## 배포

    npm run build

## dev server실행

    npm dev

## 적용스택

    webpack
    babel-polyfill, babel/core, babel/preset-env, babel-loader
    css-loader
    style-loader
    webpack-cli
    html-webpack-plugin
    sass-loader
    webpack-dev-server
